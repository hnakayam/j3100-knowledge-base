## J-3100SL

| 項目 | 仕様 |
|------|------|
| **CPU** | **メインプロセッサ**: インテル80C86-2 9.54/4.77MHzクロック相当 |
| **メモリ** | **ROM**: IPL用 32Kバイト<br>**RAM**: 標準 640Kバイト装備（最大1.5Mバイト）<br>**ビデオRAM**: 標準 グラフィックス用 32Kバイト装備 |
| **ディスプレイ表示機能** | **グラフィックス表示**: 640×400ドット 1画面<br>**画面構成**: 40字×25行<br>**文字構成**: 16×16ドット<br>**JIS第1水準、第2水準を標準装備**<br>**表示方式**: 反射型液晶ディスプレイ |
| **外部CRT接続** | **英語モード時**: CGA対応カラーCRT<br>**日本語モード時**: 使用不可 |
| **入力装置** | **キーボード**: JIS配列準拠<br>84キー（ファンクションキー、データキーの合計） |
| **補助記憶装置** | **3.5インチ フロッピーディスク装置**:<br>・2DD または 2HD型フロッピーディスクを使用<br>・720K/1.2Mバイト（フォーマッティング後）<br>・002モデル：2台装備<br>・011モデル：1台装備<br><br>**3.5インチ ハードディスク装置**:<br>・10MB/台（フォーマッティング後）<br>・011モデル：1台装備 |
| **インターフェース** | **RS-232Cインタフェース / プリンタインタフェース**:<br>・1チャンネル装備 非同期9600bpsまでサポート<br>・1チャンネル装備 セントロニクスインタフェース標準 |
| **拡張スロット** | オプションカード用スロット：1個装備 |
| **カレンダクロック** | 日付、時計機能を標準装備<br>充電型電池によるバックアップ |
| **電源** | AC100V 50Hzまたは60Hz（ACアダプタ使用） |
| **ACアダプタ使用時** | 002モデル：0.19A 14W<br>011モデル：0.25A 18W |
| **消費電力** | 内蔵バッテリ使用可能時間（完全充電状態）<br>002モデル：約5時間（FDD使用率10%時）<br>011モデル：約2時間（FDD使用率10%時・HDD使用率20%時） |
| **外形寸法** | 311（幅）×380（奥行）×72（高さ）mm |
| **重量** | 4.9kg（002モデル）、5.5kg（011モデル） |

## J-3100GT

| 項目 | 仕様 |
|------|------|
| **CPU** | **メインプロセッサ**: インテル80286-8 8MHzクロック相当 |
| **メモリ** | **ROM**: IPL用 64Kバイト<br>**RAM**: 標準 640Kバイト装備（最大2.6Mバイト）<br>**ビデオRAM**: 標準 グラフィックス用 32Kバイト装備 |
| **ディスプレイ表示機能** | **グラフィックス表示**: 640×400ドット 1画面<br>**画面構成**: 40字×25行<br>**文字構成**: 16×16ドット<br>**JIS第1水準、第2水準を標準装備**<br>**表示方式**: ガス・プラズマ・ディスプレイ |
| **外部CRT接続** | **英語モード時**: CGA対応カラーCRT<br>**日本語モード時**: 使用不可 |
| **入力装置** | **キーボード**: JIS配列準拠<br>84キー（ファンクションキー、データキーの合計） |
| **補助記憶装置** | **3.5インチ フロッピーディスク装置**:<br>・2DD または 2HD型フロッピーディスクを使用<br>・720K/1.2Mバイト（フォーマッティング後）<br>・002モデル：2台装備<br>・021モデル：1台装備<br><br>**3.5インチ ハードディスク装置**:<br>・20MB/台（フォーマッティング後）<br>・021モデル：1台装備 |
| **インターフェース** | **RS-232Cインタフェース / プリンタインタフェース**:<br>・1チャンネル装備 非同期9600bpsまでサポート<br>・1チャンネル装備 セントロニクスインタフェース標準 |
| **拡張スロット** | オプションカード用スロット：1個装備 |
| **電源** | AC100V 50Hzまたは60Hz |
| **消費電力** | 002モデル：1.3A 70W<br>021モデル：1.4A 75W |
| **外形寸法** | 311（幅）×360（奥行）×80（高さ）mm |
| **重量** | 6.6kg（002モデル）、6.8kg（021モデル） |

## J-3100GX

| 項目 | 仕様 |
|------|------|
| **CPU** | **メインプロセッサ**: インテル80286-12 12MHzクロック相当<br>**コプロセッサ**: インテル80287-12 12MHzクロック相当 |
| **メモリ** | **ROM**: IPL用 64Kバイト<br>**RAM**: 標準 1Mバイト装備（最大4Mバイト）<br>**ビデオRAM**: 標準 グラフィックス用 256Kバイト装備 |
| **ディスプレイ表示機能** | **グラフィックス表示**: 640×400ドット 1画面<br>**画面構成**: 40字×25行<br>**文字構成**: 16×16ドット<br>**JIS第1水準、第2水準を標準装備**<br>**表示方式**: ガス・プラズマ・ディスプレイ 4階調 |
| **外部CRT接続** | **英語モード時**: CGA/EGA対応カラーCRT<br>**日本語モード時**: アナログ、マルチスキャンカラーCRT |
| **入力装置** | **キーボード**: JIS配列準拠<br>98キー（ファンクションキー、データキーの合計） |
| **補助記憶装置** | **3.5インチ フロッピーディスク装置**:<br>・2DD または 2HD型フロッピーディスクを使用<br>・720K/1.2Mバイト（フォーマッティング後）<br><br>**3.5インチ ハードディスク装置**:<br>・40MB/台（フォーマッティング後） |
| **インターフェース** | **RS-232Cインタフェース / プリンタインタフェース**:<br>・1チャンネル装備 非同期9600bpsまでサポート<br>・1チャンネル装備 セントロニクスインタフェース標準 |
| **拡張スロット** | **16ビットバス / 8ビットバス**:<br>・ATフルサイズ標準カード：1個装備<br>・XTハーフサイズ標準カードまたはJ-3100専用カード：1個装備 |
| **電源** | AC100V 50Hzまたは60Hz |
| **消費電力** | 1.3A 100W |
| **外形寸法** | 370（幅）×405（奥行）×99（高さ）mm |
| **重量** | 8.5kg |
